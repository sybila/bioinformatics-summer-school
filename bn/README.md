# Boolean network analysis

This part of the repository contains Jupyter notebooks targeting Boolean network inference 
and analysis.

### Setup environment

You can setup the "minimal environment" using the following commands:

```
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

Then you can run the jupyter notebook with a kernel using this "minimal" environment:

'''
python3 -m jupyter notebook
'''

Alternatively, you can also use [CoLoMoTo docker](https://colomoto.github.io/colomoto-docker/README.html), but the tools that we will be using should be easy to install.
